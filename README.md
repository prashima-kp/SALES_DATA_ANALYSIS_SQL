# 📊 Sales Data Analysis using SQL

This project analyzes sales data using SQL to generate business insights.

## 🎯 Objectives:
- Analyze overall sales performance  
- Identify top products and regions  
- Study profit trends  
- Perform data aggregation using SQL queries  

## 🛠️ Tools & Skills:
- SQL (SELECT, WHERE, GROUP BY, ORDER BY)
- Aggregations (SUM, AVG, COUNT)
- Data analysis and interpretation  

## 📈 Key Insights:
- Top-performing products identified  
- Regional sales comparison  
- Monthly sales trends observed  
- Profit vs sales analysis performed
